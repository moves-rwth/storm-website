On this page, you can find supplementary material for the paper "[Riding the Storm in a Probabilistic Model Checking Landscape](https://doi.org/10.1007/978-3-031-75775-4_5)".

For each example, we provide Jupyter notebooks which contain the model(s) and the corresponding analysis.
All notebooks are available in the [stormpyter](https://github.com/moves-rwth/stormpyter) repository in the directory [storm-festschrift](https://github.com/moves-rwth/stormpyter/storm-festschrift).
We refer to the [README](https://github.com/moves-rwth/stormpyter/storm-festschrift/README.md) for instructions on how to run the Juypter notebooks with Docker. 


## The Random Chili Peppers
The Jupyter notebook for the music shuffle example is available in the file `random_pepper/random_pepper.ipynb`.

## Enjoyable Road Biking
The model and analysis script for the bikeriding example are available in the folder `bikeriding`.

## Scouting for Coffee Supply
The model for the coffee scouting example is available in the folder `coffeescout`.

## The Elfstedentocht Problem
The model, analysis scripts and results for the Elfstedentocht example are available in the folder `elfstedentocht`.
