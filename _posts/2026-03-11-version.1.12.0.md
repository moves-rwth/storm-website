We are happy to announce the next releases of Storm and stormpy in version 1.12.0.
<!--more-->

Storm 1.12.0 adds support for verifying interval DTMCs, extends the [DRN file format]({{ '/documentation/background/drn.html' | relative_url }}), and adds improvements in multi-objective model checking and verification of parametric Markov models.
We refer to the [release notes of Storm](https://github.com/stormchecker/storm/releases/tag/1.12.0) for all details.

The [stormpy](https://stormchecker.github.io/stormpy/) Python bindings are also released in a new version 1.12.0 that comes with improved support for interval models, extended support for the JANI interchange format and for Markov automata.
We refer to the [release notes of stormpy](https://github.com/stormchecker/stormpy/releases/tag/1.12.0) for all details.


You can get the new release of Storm either by building from [source]({{ '/documentation/obtain-storm/build.html' | relative_url }}), via [Homebrew]({{ '/documentation/obtain-storm/homebrew.html' | relative_url }}) or by using a [Docker container]({{ '/documentation/obtain-storm/docker.html' | relative_url }}).
You can obtain the stormpy Python package by simply executing `pip install stormpy`.

If you experience any bugs, please [let us know]({{ '/documentation/obtain-storm/troubleshooting.html#file-an-issue' | relative_url }}).
